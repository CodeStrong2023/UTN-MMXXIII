Secuencias de Escape
\' = Mostrar una comilla simple dentro de una cadena en la salida de infomarción
\" = Mostrar doble comilla dentro de una cadena en la salida de infomarción
\\ = Barra invertida, diagonal inversa, es para mostrar una back slash
\n = Salto de línea
\r = retorno
\t = Tabulacion
\b = Retroceso
\f = Salto de pagina

Operador de Decremento
